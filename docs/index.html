أدوات المطوّرين من مستوى الحاسوب على هاتفك

Edge Console أداةُ تطويرٍ موسّعةٌ تعمل ضمن متصفّح Microsoft Edge، صنعتها sbay-dev — مطورو المستقبل. تمنحك تجربةَ DevTools كاملةً على الشاشات الكبيرة المحمولة، مع جسرٍ أصليٍّ إلى gh copilot cli عبر بروتوكول MCP.

    المزايا
    
واجهةٌ من زمنٍ آخَر، تعمل اليوم

    
كلّ مكوّنٍ صُمِّم ليبدو كأنّه قادمٌ من نسخة Edge المستقبليّة — بأدوات مطوّرٍ كاملةٍ على الشاشات اللوحيّة المحمولة.

  

  


    

      

        
      

      
كونسول كامل

      
يلتقط console.* والاستثناءات وفشل الشبكة عبر CDP، مع تلوينٍ ومرشّحاتٍ ومستوياتٍ احترافية.

      متاح الآن
    


    

      

        
      

      
شجرة Pofs

      
استعراضٌ كاملٌ لملفّات الموقع الفعّال وملفّات WasmMvc (OPFS · IndexedDB · SQLite) كشجرةِ مجلّدات، مع زرّ تحميل لكلّ عقدة.

      قريبًا · v1.1
    


    

      

        
      

      
الشبكة

      
مراقبةٌ مباشرةٌ للطلبات، توقيتاتٌ، رؤوسٌ، وحمولاتٌ — كلّها قابلةٌ للنقل إلى Copilot كسياقٍ بضغطة.

      على الخارطة
    


    

      

        
      

      
العناصر

      
شجرة DOM حيّة، تعديلٌ آنيٌّ للأنماط، ومحرّك styles باللمس — مهيّأٌ للشاشات الكبيرة.

      على الخارطة
    


    

      

        
      

      
جسر Copilot عبر MCP

      
خادم MCP أصليٌّ يبثّ الأحداث إلى gh copilot cli ليصبح المساعدُ مدركًا لحالة الصفحة وأخطائها لحظةً بلحظة.

      متاح الآن
    


    

      

        
      

      
واجهة WasmMvc

      
واجهةُ DevTools نفسُها مكتوبةٌ بـ C# / Razor فوق NetWasmMvc.SDK — أداءٌ أصليٌّ وتشغيلٌ آمنٌ في sandbox.

      متاح الآن
