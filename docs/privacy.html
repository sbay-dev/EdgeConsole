الملخّص ما نجمعه الأذونات تكامل Copilot المشاركة مع الغير حقوقك التواصل

الملخّص

إضافة sbay-dev · Edge Console أداةُ تطويرٍ تعمل بالكامل على جهازك. لا نُشغّل خوادمَ خلفيّةً، ولا نُرسل بياناتِ تصفّحك إلى أيّ طرفٍ ثالث، ولا نستخدم أنظمةَ تحليلٍ أو إعلاناتٍ أو متعقّبات.

كلّ ما تلتقطه الإضافة من معلوماتٍ هندسيّةٍ (سجلّات الكونسول، أحداث الشبكة، عناصر DOM، ملفّات الموقع) يبقى محلّيًّا في متصفّحك أو يُمرَّر — باختيارك — إلى أداتك المحلّيّة gh copilot cli عبر جسرِ MCP.

ما نجمعه (وما لا نجمعه)

تعمل الإضافة على بياناتٍ مؤقّتةٍ ضمن ذاكرة المتصفّح فقط، وتشمل:

  • سجلّات الكونسول والاستثناءات للصفحة الفعّالة عند فتح DevTools.
  • أحداث الشبكة (URL، رؤوس، توقيتات، رموز الحالة) من تبويبك الحاليّ.
  • لقطاتٌ من شجرة DOM والأنماط عند طلبك صراحةً.
  • قائمةُ ملفّات الموقع الفعّال وقواعد البيانات الخاصّة بـ WasmMvc (في الإصدارات اللاحقة).

لا نجمع: كلمات المرور، رؤوس الاستيثاق، الكوكيز، البيانات الشخصيّة، السجلّات الجغرافيّة، أو أيّ معرّفٍ يُحدّد هويّتك. كلّ ما سبق يبقى محلّيًّا ولا يُرسَل إلى مُلّاك الإضافة.

الأذونات وسبب طلبها

  • debugger · للوصول إلى Chrome DevTools Protocol وقراءة أحداث الكونسول والشبكة.
  • storage · لحفظ تفضيلاتك المحلّيّة (السمة، حالة المرشّحات).
  • scripting + activeTab · لحقن الواجهة المضمّنة عند طلبها فقط.
  • nativeMessaging · للتواصل مع جسر sbay-devtools-bridge الذي يعمل على جهازك.
  • host_permissions: <all_urls> · لأنّ DevTools تحتاج التوصيل بأيّ موقع تعمل عليه أنت.

تكامل GitHub Copilot عبر MCP

حين تُفعّل جسرَ Copilot، يَصِل خادم MCP المحلّيّ (يعمل على جهازك) بـ gh copilot cli. عند هذه النقطة، ما تطلبه أنت من Copilot قد يُرسَل إلى GitHub وفقًا لسياسة Copilot الخاصّة بك. الإضافة لا ترسل أيّ شيءٍ إلى GitHub بشكلٍ مباشر.

يمكنك تعطيلُ الجسر في أيّ وقتٍ بإيقاف العمليّة المحلّيّة، وستستمرّ بقيّةُ الأدوات بالعمل.

المشاركة مع أطرافٍ ثالثة

لا نبيع، لا نؤجّر، ولا نشارك بياناتك. لا تتصل الإضافة بأيّ خدمةٍ خارجيّةٍ تابعةٍ لنا. الاتصالات الوحيدة الخارجة هي تلك التي يبدؤها gh copilot cli بناءً على طلبك أنت.

حقوقك

بما أنّنا لا نخزّن بياناتك على خوادمنا، فإنّ التحكّم الكامل بيدك: تعطيل الإضافة، إزالتها، أو مسح تخزينها المحلّيّ من edge://extensions يحذف كلّ شيءٍ فورًا.
